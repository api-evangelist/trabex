# Trabex

Trabex is a trade compliance platform that provides automated export compliance, shipment management, restricted party screening, and Automated Export System (AES) filing services. Trabex offers APIs for integrating trade compliance workflows into logistics and ERP systems.

**URL:** [apis.yml](https://raw.githubusercontent.com/api-evangelist/trabex/refs/heads/main/apis.yml)

## APIs

### Trabex Trade Compliance API

The Trabex Trade Compliance API provides programmatic access to export compliance automation including shipment data submission, export documentation generation, Automated Export System (AES) filing, and restricted party screening (RPS).

**Human URL:** [https://apidocs.trabex.io/](https://apidocs.trabex.io/)
